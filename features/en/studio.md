# Editing Studio

> Edit and customize your videos directly in your browser.
> Add text, logo, filters… no software required.

---

## What is it for?

You have a translated video and want to polish it before publishing:

- **Add your logo** or watermark
- **Insert text** — titles, captions, quotes
- **Blur a face** automatically
- **Add speech bubbles** comic-book style
- **Cut and rearrange** clips
- **Export** in the right format for each platform (16:9, 9:16, 1:1)

## How does it work?

**1. Open a video** in the Studio from your library

**2. Pick a tool** from the toolbar (16 tools available)

**3. Place it** on the timeline — drag, drop, resize

**4. Preview** the result in real time

**5. Export** — the final video is generated with all changes

## Available tools

| Tool | Description |
|------|-------------|
| Text | Add titles, captions, subtitles |
| Watermark | Embed your logo or brand |
| Face blur | Automatically blur faces |
| Speech bubble | Comic-book style |
| PIP | Picture-in-Picture |
| QR Code | Generate and embed a QR code |
| Timer | Visible countdown in the video |
| + 9 more | Filters, shapes, transitions… |

## Export formats

- **16:9** — YouTube, Vimeo
- **9:16** — TikTok, Instagram Reels, YouTube Shorts
- **1:1** — Instagram, Facebook
- **MP4** with or without burned-in subtitles
