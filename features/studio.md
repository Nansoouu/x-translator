# Studio d'édition

> Montez, éditez et personnalisez vos vidéos directement dans le navigateur.  
> Ajoutez du texte, un logo, un filtre… sans aucun logiciel.

---

## Ça sert à quoi ?

Vous avez une vidéo traduite et vous voulez la peaufiner avant de la publier :

- **Ajouter votre logo** ou watermark sur toute la vidéo
- **Incruster du texte** — titres, légendes, citations
- **Masquer un visage** avec un flou
- **Ajouter une bulle de parole** façon bande dessinée
- **Couper, réorganiser** les passages
- **Exporter** au format adapté à chaque plateforme (16:9, 9:16, 1:1)

## Comment ça marche ?

**1. Ouvrez une vidéo** dans le studio depuis votre bibliothèque

**2. Choisissez un outil** dans la barre d'outils (16 outils disponibles)

**3. Positionnez-le** sur la timeline — glissez, déposez, redimensionnez

**4. Prévisualisez** le résultat en temps réel

**5. Exportez** — la vidéo finale est générée avec tous vos changements

## Outils disponibles

| Outil | Description |
|-------|-------------|
| Texte | Ajoutez un titre, une légende, un sous-titre |
| Watermark | Incrustez votre logo ou marque |
| Flou visage | Masquez automatiquement les visages |
| Bulle parole | Style bande dessinée |
| PIP | Image dans l'image (Picture-in-Picture) |
| QR Code | Générez et incrustez un QR code |
| Chronomètre | Compteur visible dans la vidéo |
| + 9 autres | Filtres, formes, transitions… |

## Formats d'export

- **16:9** — YouTube, Vimeo
- **9:16** — TikTok, Instagram Reels, YouTube Shorts
- **1:1** — Instagram, Facebook
- **MP4** avec ou sans sous-titres incrustés
